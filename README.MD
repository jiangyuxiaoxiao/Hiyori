## 特色
+ 使用 Nonebot2 框架 + Go-Cqhttp QQ协议端 + OnebotV11 规范 
+ 丰富的管理功能、Bot特性支持：
  + 插件开关
  + 插件菜单生成
  + 黑名单管理
  + 定时撤回
  + 调用冷却
  + 自定义物品开发
  + 多CQHTTP客户端连接支持：
    + 多Bot在同一群聊时不重复处理事件，自动选择指定Bot进行事件处理
    + 多Bot在同一群聊时，处理Bot下线后自动切换到备用Bot
